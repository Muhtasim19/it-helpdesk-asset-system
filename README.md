# IT Help Desk and Asset Management System

## Overview
A web-based platform for managing IT support tickets and tracking hardware and software assets.

## Team
- Muhtasim Haq – Frontend Developer (React, UI/UX)
- Sabirul – Backend & Database Developer (FastAPI, PostgreSQL)

## Tech Stack
### Frontend
- React
- Vite
- Tailwind CSS
- Axios
- React Router

### Backend
- Python
- FastAPI
- SQLAlchemy
- PostgreSQL
- JWT Authentication

### Tools
- Git & GitHub
- VS Code
- Postman

## Features
### Asset Management
- Add, edit, and delete assets
- Asset categories and statuses
- Assign assets to users
- Warranty tracking

### Help Desk
- Create support tickets
- Assign technicians
- Ticket priorities and statuses
- Comments and updates

### Dashboard
- Open tickets overview
- Asset statistics
- Reports and charts
